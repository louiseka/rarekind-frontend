## Project Overview
**RareKind** is a frontend **React** application that helps users browse and manage collections and items. Users can explore public collections, create and edit their own collections, and manage items within those collections.  

The application interacts with the **RareKind backend API**, uses **Redux Toolkit** for state management, and **CSS Modules** for modular and maintainable styling.

## Features

### User Authentication
- Login with persistent sessions  
- Protected routes for authenticated users  

### Collections Management
- Fetch collections from the backend  
- View individual collections and items  
- Add, edit, and delete collections and items  
- Manage user-specific collections  

### State Management & Styling
- **Redux Toolkit** for global state (authentication, collections)  
- **CSS Modules** for component-scoped styling

## Installation (Local Development)
Follow these steps to set up RareKind on your local machine:

```bash
# Clone the repository
git clone https://github.com/your-username/rarekind-frontend.git

# Navigate into the project directory
cd rarekind-frontend

# Install dependencies
npm install

# Set up environment variables
Create a .env file in the root directory and add the necessary variables (e.g., backend API URL, auth keys).

# Start the backend server
npm run dev
```
